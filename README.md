# 𝄞Make an Orchestra 
Allow users print music via different instruments using an AI model.

### 📋Aims of App
- Allow users to select instruments, tones, keys etc.
- Allow users to print music on frontend
  
### 👩‍💻Technical Details

- React
- Next.js
- Shadcn/UI
- Zod
- Recharts
